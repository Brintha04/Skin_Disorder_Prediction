# Skin Disorder Prediction Project

## Overview
This project aims to develop a predictive model for identifying various skin disorders based on a dataset containing labeled skin images. It utilizes machine learning techniques to classify skin conditions and provide insights into skin health.

## Dataset Description
The dataset consists of images of skin lesions, along with their corresponding labels indicating the type of skin disorder. The key attributes of the dataset include:
- **Image ID**: Unique identifier for each image.
- **Image URL**: The location of the image file.
- **Diagnosis**: The label indicating the skin disorder (e.g., Melanoma, Benign, etc.).

## Project Objectives
- To preprocess and clean the dataset for effective model training.
- To explore various machine learning algorithms for classification.
- To evaluate the model performance using relevant metrics such as accuracy, precision, recall, and F1 score.
- To provide a user-friendly interface for users to upload skin images and receive diagnostic predictions.

## Technologies Used
- Python
- Libraries: TensorFlow, Keras, pandas, scikit-learn, OpenCV, Flask
- Jupyter Notebook for experimentation
- Git for version control
